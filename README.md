# cyanotypegenerator

  a browser-based tool for applying an authentic cyanotype darkroom effect to any [photo](https://elizwangwastaken.github.io/cyanotypegenerator/)

# how the processing works

the processor converts your image to greyscale using perceptual luminance weights, applies an S-curve to push shadows and lift highlights, inverts the tones, then screen0blends Prussian blue (#0b2545), mirroring the iron-salt chemistry of a traditional cyanotype print
